Algerian ID & Passport Reader - Kotlin Example
A new Kotlin example project for reading ID card and passport of Algeria using algerianidpassportreader SDK.

https://github.com/user-attachments/assets/8020629c-8e52-48cf-a479-698d805dabd1

## Getting Started
1 - Add Your Token :
Go to app/src/main/java/com/algerianidpassportreader/testapp/MainActivity.kt and enter your token in the initialize() function:

AlgerianIDPassportSDKManager.initialize(
    context = this@MainActivity,
    token = "YOUR_TOKEN_HERE"  // ‚Üê Enter your token here
)

2 - Connect your Android device to your PC

3 - Open in Android Studio and run the app
